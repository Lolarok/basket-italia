# 🏀 Basket Italia - WordPress Theme

Portale italiano di notizie su basket: NBA, Serie A, EuroLega.

## Design
- Colori: Arancio #E8821A + Oro #FFD700 su sfondo nero
- Font: Oswald (titoli) + Roboto Condensed
- Features: Score ticker, statistiche, classifiche

## Fonti RSS Basket
- Gazzetta dello Sport Basket
- Corriere dello Sport Basket
- ESPN NBA
- BleacherReport NBA
- EuroLeague
- BasketNews
